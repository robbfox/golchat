# WhatsApp Chat Replay

This is a [Next.js](https://nextjs.org) project that recreates a chat interface from a text file source.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Structure

- `src/app`: Application pages and layout.
- `src/components`: UI components (ChatBubble, ChatContainer, Header).
- `src/data.json`: Parsed chat data.
- `chat.txt`: Original source text.

## Deployment

The project is configured for deployment on Vercel.
